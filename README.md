## 🌟 About

**Hetro Movie** is a sleek and fast cross-platform movie discovery application built with **Flutter** and powered by the **TMDB API**. It gives users a clean and intuitive way to browse thousands of movies, search for their favorites, and explore by category — all from one beautiful app.

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) `>= 3.0.0`
- A [TMDB API Key](https://www.themoviedb.org/settings/api) (free)
- Android Studio or VS Code

```bash
flutter doctor
```

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/Hyzinx/hetro_movie.git

# 2. Navigate into the project
cd hetro_movie

# 3. Install dependencies
flutter pub get
```

### Add Your TMDB API Key

Find the API config file in `lib/` and replace with your key:

```dart
const String apiKey = 'YOUR_TMDB_API_KEY_HERE';
```

> Get your free API key from [themoviedb.org](https://www.themoviedb.org/settings/api)

### Run The App

```bash
# Android / iOS
flutter run

# Web
flutter run -d chrome

# Windows
flutter run -d windows

# Build APK
flutter build apk --release
```
